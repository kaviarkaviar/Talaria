# Workouts

Put any Intervals.icu workout `.txt` or `.md` files here.

Use filenames like:

```text
YYYY-MM-DD_easy-bike-45m.txt
YYYY-MM-DD_swim-technique.txt
YYYY-MM-DD_strength-upper-body.txt
YYYY-MM-DD_yoga-mobility.txt
```

The app infers the workout type from the filename, title, and workout text.

See the `examples` folder for bike, run, swim, strength, yoga, walk, row, and single-file Markdown plan examples.
